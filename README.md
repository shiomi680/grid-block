# Grid Block

obsidian のグリッドレイアウトを作成するプラグインです。
grid-block という名前のコードブロックを作成すると、簡単に gird レイアウトを作成できます。
---で区切られたブロックがそれぞれのグリッドになります。
一番上の行に各ブロックのグリッドサイズを指定します。
グリッドは mui の Grid を使用しています。
https://mui.com/material-ui/react-grid2/

## 使い方

```grid-block
6,6
---
# コンテンツ１
- リスト
- リスト
- リスト

---
# コンテンツ２
- リスト
- リスト
- リスト
```
